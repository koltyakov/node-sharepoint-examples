# SPNode Exapmles

## Clone the repo

```bash
git clone https://gitlab.com/spnode-public/sp-request-example-01 && cd sp-request-example-01
```

## Restore dependencies

```bash
npm install
```

## Examples

### Run with typescript

#### sp-request

```bash
npm run ts:run:request
```

#### sp-pnp-node

```bash
npm run ts:run:pnp
```

#### sp-jsom-node

```bash
npm run ts:run:jsom
```

## Build

### Linting

```bash
npm run tslint
```

### Transpile

```bash
npm run build
```

### Debug

In VS Code, open a .ts to debug, place brake point and hit `F5`.
